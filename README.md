# PplSafer

APK Malware Analysis Tool using machine learning and static analysis.

## Quick Start

```bash
git clone https://gitlab.com/sam-mg/pplsafer
cd pplsafer
./run.sh
```

Navigate to `http://localhost:5001` and upload your APK file for analysis.
